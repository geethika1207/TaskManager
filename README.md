# Task Manager Web App
A simple and clean task manager built using HTML, CSS, and JavaScript.
This Frontend Web  app helps users organize tasks by adding due dates, tracking completion status, and managing tasks efficiently with a minimal interface.
 Live Demo :
 https://verdant-marzipan-40461c.netlify.app/
 # Features
 1. Add tasks with a due date
 2. Automatically sorts tasks by date
 3. Mark tasks as Completed or Pending
 4. Visual color change based on task status
 5. Delete individual tasks with confirmation
 6. Delete all tasks at once with confirmation
 7. Tasks are stored using Local Storage (data remains after refresh)
 8. Clean and responsive UI
# How It Works
 1. Enter a task name
 2. Select a due date using the calendar
 3. Click Add Task
 4. A task card is created showing:
    1.Task name
    2.Status (Pending / Completed)
    3.Due date at the bottom
 5. Tasks are automatically arranged based on the selected date
# Technologies Used
 1.HTML – structure
 2.CSS – styling and layout
 3.JavaScript – logic, DOM manipulation, and local storage
No backend or database is used. This is a front-end focused project.
